# ARM (Հայերեն մինի ծրագրավորման լեզու)

ARM-ը փոքր ծրագրավորման լեզու է, որը թույլ է տալիս գրել ծրագիր հայերեն բառերով։ Այն աշխատում է `.arm` ֆայլերը Python-ի թարգմանելով և անմիջապես կատարելով։

---

## Պահանջներ

* Python 3
* UTF-8 

---

## Ֆայլեր

* `arm.py` — թարգմանիչ և գործարկիչ
* `example.arm` — հայերեն ծրագիր

---

## Ծրագրի ստեղծում

Ստեղծեք ֆայլ `.arm` ընդլայնմամբ։

Կարող եք օգտագործել հայերեն բառեր փոփոխականների, տողերի և հրամանների համար։

Օրինակ․

```
ամբողջ ա = 10
տող տող1 = "Բարև աշխարհ"

եթե ա > 5:
    տպել(տող1)
այլապես:
    տպել("Փոքր է")
```

---

## Հիմնական հրամաններ

* `եթե` → if
* `այլապես` → else
* `մինչ` → while
* `տպել` → print
* `ճշմարիտ` → True
* `կեղծ` → False

Տիպերի բառերը (`ամբողջ`, `տող`, `բուլյան`) պարտադիր չեն և օգտագործվում են միայն ընթեռնելիության համար։

---

## Ծրագրի գործարկում

Տերմինալում գրեք՝

```
python arm.py armenian.arm
```

Ծրագիրը կթարգմանվի Python-ի և կկատարվի

---

## Նշումներ

* Python-ի կանոնները պահպանվում են (նահանջները կարևոր են)
* Հայերեն փոփոխականների և ֆունկցիաների անունները լիովին աջակցվում են
